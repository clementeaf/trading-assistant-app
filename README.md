# Trading Assistant App

Aplicación de asistente para trading.

## Descripción

Esta aplicación proporciona herramientas y funcionalidades para asistir en operaciones de trading.

## Instalación

```bash
cd backend
pip install -r requirements.txt
```

## Uso

```bash
cd backend
uvicorn main:app --reload
```

La aplicación estará disponible en `http://localhost:8000`

## Documentación

Una vez que la aplicación esté corriendo, puedes acceder a:
- Documentación interactiva: `http://localhost:8000/docs`
- Documentación alternativa: `http://localhost:8000/redoc`
